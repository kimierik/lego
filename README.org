* How to use/start
** manually
start the backend with by goind to the backend dir and running
#+begin_src bash
go run .
#+end_src

then navigate to the frontend directory and run
#+begin_src bash
trunk serve
#+end_src

make sure you have [[https://trunkrs.dev][trunk]] installed
